Participant Information

Before analysing the experimental data, we examine whether participants’ self-assessed English proficiency correlates with their LexTALE scores. LexTALE is used as an objective measure of English vocabulary knowledge, while the self-assessment scores reflect participants’ perceived language abilities in four domains:

  • speaking
  • listening
  • reading
  • writing

Correlation between LexTALE and Self-Assessment

The following data frame contains the LexTALE results and the self-assessment scores for each participant.

Visualization of the Correlations

To visualise the relationship between LexTALE scores and self-assessed language skills, the data are transformed into long format and plotted separately for each skill.

# Transform dataframe to long format
corr_df <- participant_df %>%
  pivot_longer(
    cols = Speaking:Writing,
    names_to = "skill",
    values_to = "score"
  )

# Plot
p <- ggplot(corr_df,
       aes(x = lex_tale_result, y = score)) +
  geom_point() +
  geom_smooth(method = "lm", se = TRUE, color = "mediumpurple1", fill = "mediumpurple1") +
  facet_wrap(~skill) +
  theme_minimal() +
  labs(x = "LexTALE (%)", y = "Self-Assessment")

p

# Save
ggsave(
  "lextale_correlations.png",
  plot = p,
  width = 8,
  height = 6
)

Reaction Time Filtering

To remove implausible reaction times, a cutoff is applied.

Only trials with reaction times between 200 ms and 2000 ms are retained.

This removes extremely fast responses (likely accidental key presses) and extremely slow responses that may reflect distractions or interruptions.

Inferential Statistics

To formally test the research questions, mixed-effects models are used.

Mixed-effects models are appropriate for experimental data because they allow us to account for variability across both:

  • participants
  • stimuli

Data Loading and Preparation

  • Data loading: The full experimental dataset containing reaction times and accuracy values as well as participant data is loaded from a CSV file.
  • Filtering: everything that was removed from the above dataset needs to be removed here as well (practice items, nonwords, certain participants)
  • Variable coding: Before fitting the models, variables are converted to appropriate types (e.g., numeric or factor). The ordering of factor levels is also specified for interpretability.
# Load dataset
inf_df <- read_csv("LDT_longlist_merged_2026_01_09.csv", show_col_types = FALSE, col_types = cols(
                          lex_tale_result = col_character()
                        )) %>%
  mutate(lex_tale_result = as.numeric(str_replace(lex_tale_result, ",", ".")))    # make sure LexTALE gets parsed right

# Filtering
prepped_df <- subset(inf_df, Item_type == "exp" & Condition == "word" & !(Participant %in% c("2400")) & RT >= 200 & RT <= 2000)

# Variable coding
prepped_df <- prepped_df %>%
  mutate(
    Accuracy = as.numeric(Accuracy),
    Participant = as.factor(Participant),
    Emotional_valence_cat = factor(Emotional_valence_cat, levels = c("neutral", "positive", "negative")),
    Length = as.numeric(Length),
    RT = as.numeric(RT),
    Stimulus = as.factor(Stimulus),
    lex_tale_result = as.numeric(lex_tale_result),
  )

Standardizing Continuous Predictors

Continuous predictors are standardized (z-transformed). This improves model convergence and ensures that predictors are on comparable scales.

prepped_df <- prepped_df %>%
  mutate(
    lex_tale_z = scale(lex_tale_result),
    Length_z   = scale(Length)
  )

Accuracy Analysis

Accuracy is analysed using a generalized linear mixed-effects model with a binomial link function.

The model includes:

Fixed effects:

  • LexTALE score (standardized)
  • emotional valence
  • word length

Random effects:

  • random intercepts for participants
  • random intercepts for stimuli
# Accuracy mixed model
acc_model <- glmer(Accuracy ~ lex_tale_z * Emotional_valence_cat + Length_z + (1| Participant) + (1| Stimulus), family = binomial, data = prepped_df, control = glmerControl(optimizer = "bobyqa", optCtrl = list(maxfun = 1e5)))

Reaction Time Analysis

Reaction times are analysed using a linear mixed-effects model.

Only trials with correct responses are included, as incorrect trials may reflect different cognitive processes.

The model includes:

Fixed effects:

  • LexTALE score (standardized)
  • emotional valence
  • word length

Random effects:

  • random intercepts for participants
  • random intercepts for stimuli

Interpretation

The results of the mixed-effects models are interpreted with respect to the research questions:

  1. Does English proficiency influence the accuracy or speed of lexical decisions?

  2. Do emotionally valenced words (positive or negative) show a processing advantage compared to neutral words?

  3. Does emotional valence interact with English proficiency in predicting reaction times or accuracy?

The answers to these questions are based on the inferential statistical results reported above.
